# Contributing to Zendvo

Thank you for your interest in contributing to Zendvo!

## Code of Conduct
By participating, you agree to follow our [Code of Conduct](./CODE_OF_CONDUCT.md).

## How to Contribute
1. Fork the repo and create your feature branch.
2. Ensure your code follows the established Next.js and TypeScript standards.
3. Verify that your changes do not break existing blockchain or database interactions.
4. Open a Pull Request with a clear explanation of your improvements.

## Development Setup
Refer to the [Quick Start section in the README](../README.md#quick-start) for setup instructions.
