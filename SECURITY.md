# Security Policy

## Reporting a Vulnerability
We prioritize the security of our users and their financial gifts. If you find a security vulnerability, please report it privately.

### How to Report
Please do NOT open public issues for security vulnerabilities. Send an email to the project maintainers.

### Security Best Practices
- **Smart Contracts**: Our time-locking logic is handled by audited Stellar Soroban smart contracts.
- **Data Encryption**: Sensitive user data and tokens are encrypted at rest and in transit.
- **Verification**: We use phone-based verification and secure session management to protect user accounts.
